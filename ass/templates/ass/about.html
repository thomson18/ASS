About us

AUTOMATED SECURITY SYSTEM

The Automated security system is a system which automates the security monitoring process and enables user friendly security processes. It detects and recognizes the human faces in the live stream and alerts the user or owner with the proper list of visitors. It connects the user and model with a TCP connection. This system also provides us with a storage system which is Firebase Realtime Database facilitates to store the details and the time of the appearances of the visitors. This system has a frontend web page which provides us more flexibility to operate and use the model. It provides us a detailed excel sheet of the students in the perspective of attendance management.

The purpose for the development of this system is to make the security monitoring process automate where we do not need a person to secure our place. For the security purpose we need a security person there in our place and we have to pay for him. But using this system we can make the above process automate using this system. No need to pay money and this model predicts us with accurate results and sends us alerts. To automate the attendance process in any organization.

The beauty of this is that it can run in any device or computer with minimum RAM it doesn’t require any GPU’s or any high performance system. It can run on a normal machine for example it can be used in a way that takes input from a normal camera that is live stream and we can get a list of the persons who are in that place. For example in colleges or universities or any educational institutions the traditional way of taking attendance of the staff is taking their signature ,in this we can use this project and take the attendance of the staff and we can make a note of it. And we can do this using raspberry pi also. Raspberry pi is nothing but a computer without any input and output devices and contains ports for connecting them and has a small amount of ram generally 1GB,2GB,4GB for new raspberry pi 4. It can also be used in organizations where only those members have to be there .For surveillance purposes in the organization .In some companies, in some places there shouldn’t be outside workers, for example in the research and development sector no outside members are allowed we can use this there also. It has a wide range of applications due to its simplicity.
